## 소통하는 개발자 안현지입니다 👋

  사용자 중심 소통을 위한 <code>직관적이고 친근한 UI 디자인</code>으로, 이용자들에게 <code>더 나은 경험</code>을 선사하는 <code>프론트엔드 개발자</code>입니다. 

  <code>사용자 편의성</code>을 최우선으로 생각하여, IT 서비스의 본연의 임무를 완수하며, <code>사람과 사람 사이</code>를 유기적으로 연결합니다.
